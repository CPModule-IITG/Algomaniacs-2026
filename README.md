# Algomaniacs-2026

Contains all the lecture materials of Algomaniacs 2026.

Lecture summary:

## Lecture 1 (Square Root Decomposition)

* Discussion on Square Root Decomposition and Mo's Algorithm.
* Lecturer : Vaibhav Singh
* Discussed about how different problems can get boiled down to the simple Mo's algorithm's template.

## Lecture 2 (Mathematics)

* Discussion on various mathematical topics relevant to competitive programming.
* Lecturer : Harsh
* Topics covered include Binary Exponentiation, Modular Inverse, Combinatorics, Sieve of Eratosthenes, Euclidean Algorithm, Divisor properties, Möbius Function, Matrix Exponentiation, and Catalan Numbers.
