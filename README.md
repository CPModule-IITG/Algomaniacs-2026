# Algomaniacs-2026

Contains all the lecture materials of Algomaniacs 2026.

Lecture summary:

## Lecture 1 (Square Root Decomposition)

* Discussion on understanding and implementing Square Root Decomposition and Mo's Algorithm.
* Lecturer : Vaibhav Singh
* Topics covered include Square Root Decomposition, Mo's Algorithm, and reducing various problems to Mo's algorithm's template.

## Lecture 2 (Mathematics)

* Discussion on various mathematical topics relevant to competitive programming.
* Lecturer : Harsh
* Topics covered include Binary Exponentiation, Modular Inverse, Combinatorics, Sieve of Eratosthenes, Euclidean Algorithm, Divisor properties, Möbius Function, Matrix Exponentiation, and Catalan Numbers.

## Lecture 3 (Fast Fourier Transform)

* Discussion on transitioning from O(N²) math to O(N log N) code using FFT.
* Lecturer : Shreyans Garg
* Topics covered include Fast Fourier Transform (FFT), Polynomial Multiplication, Complex Root of Unity, and convolution.

## Lecture 4 (Segment Trees)

* Discussion on understanding and implementing Segment Trees.
* Lecturer : Shashwat Shukla
* Topics covered include Segment Tree Construction, Point Updates, Range Queries, Generalization techniques, and Lazy Propagation for range updates.
